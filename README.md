# IBM-Cloud-Functions-tutorial

# Desarrollo de una aplicación web usando MicroServicios implementados como FaaS (Serverless) en IBM Cloud
El objetivo de este tutorial es enseñar a desarrollar MicroServicios utilizando la tecnología FaaS (Function as a Service) 
en la plataforma de IBM Cloud. En este ejercicio crearemos microservicios en JavaScript y en Swift 4.2

El texto del tutorial se encuentra en el [wiki](https://github.com/huibert7/IBM-Cloud-Functions-tutorial/wiki)

Adicionalmente, hay un manual en formato PDF en el directorio Documentation.
